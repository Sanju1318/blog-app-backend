Blog Application Backend

A robust backend application built using Spring Boot that provides secure authentication and authorization for a blogging platform. The application uses JWT-based authentication and Role-Based Access Control (RBAC) to manage user access and permissions.

Features
JWT Authentication and Authorization
Role-Based Access Control (Admin/User)
User Registration and Login
CRUD Operations for Blog Posts
Comment Management System
Post Retrieval by ID
User-Specific Data Access
File Upload Support
RESTful API Architecture
PostgreSQL Database Integration
Exception Handling and Validation
Tech Stack
Java
Spring Boot
Spring Security
JWT (JSON Web Token)
Spring Data JPA
PostgreSQL
Maven
REST APIs
Key Functionalities
Secure user authentication using JWT.
Role-based access management for different users.
Create, update, delete, and retrieve blog posts.
Add and manage comments on posts.
Retrieve post details using Post ID.
Store and manage user-related data in PostgreSQL.
Handle file uploads associated with users and posts.
Database

PostgreSQL is used as the primary database for storing users, posts, comments, roles, and related application data.
